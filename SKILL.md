목적
- 다수의 사용자가 공동 목표를 가지고 챌린지를 진행하도록 도와주는 앱

주요 기능
- 챌린지 생성: 사용자가 새로운 챌린지를 만들 수 있도록 지원
- 챌린지 참여: 사용자가 기존 챌린지에 참여할 수 있도록 지원
- 매일 챌린지 진행 여부 기록 : 참여자가 매일 챌린지를 진행했는지 여부를 기록할 수 있도록 지원

상세 기능
- 챌린지 진행 여부는 사진 혹은 글을 올리도록 함
- 참여자 별 챌린지 진행 여부가 체크된 캘린더 혹은 기록판이 있음
- 각 참여자는 챌린지 기간동안 챌린지를 수행한 날 사진 혹은 글을 올려 진행 여부를 표시함
- 전체 유저가 서로의 진행 여부를 볼 수 있도록 함
- 챌린지 별 지켜야 하는 챌린지 진행 횟수가 있으며, 해당 조건을 충족하지 못한 기간이 있을 경우 벌금 책정
